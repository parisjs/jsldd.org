# JSLDD

This the [docpad][] site of [jsldd.org][].

## Install

    npm install
    bower install

## Usage

    docpad run

And then point your browser to http://localhost:9778/.

## Deploy

Generate the static site:

    docpad deploy-ghpages --env static
